Enhanced Homomorphic Encryption Technique using Extended Euclidean Algorithm

Abstract

Confidential information transmission over the internet is a significant challenge, with cryptographic algorithms being the best choice for protection. Securing information over the cloud is another major challenge. Homomorphic encryption is a promising method for securing information in the cloud. This project aims in proposing RSA algorithm using multiple public key pairs with Homomorphic Encryption, which generates a key pair from multiple keys using RSA homomorphic encryption. This technique uses mathematical logic to obtain the public key directly, stopping attacks on finding the private key. The Extended Euclidean Algorithm (EEA) is used for generating keys, following the RSA Homomorphic encryption technique. Hence,the algorithm is fast and secure, utilizing a modulo operator, and utilizing Homomorphic operations to enhance security for confidential information in the cloud.
